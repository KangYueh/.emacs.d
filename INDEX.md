## 🔑 Essential Information

### Top 10 Keybindings (Memorize These)
```
C-c n c     Capture new note
C-c n f     Find note
C-c n i     Insert link
C-c n l     Toggle backlinks
C-c n j     Daily note
C-c n D     Daily review
C-c n P     Process fleeting notes
C-c n g     Graph view
C-c n S     Statistics
C-c a       Agenda
```
**Full list:** [QUICK_REFERENCE.md](QUICK_REFERENCE.md)

### Daily Workflow (10 minutes)
```
Morning (2 min):
  C-c n j → Daily note
  Write intentions

Evening (8 min):
  C-c n D → Daily review
  C-c n P → Process fleeting
  C-c n O → Check orphans
```
**Detailed guide:** [ZETTELKASTEN_GUIDE.md](ZETTELKASTEN_GUIDE.md) → Workflow

### Note Types
```
Fleeting    → Quick captures (process within 1-2 days)
Permanent   → Refined atomic notes (core of system)
Concept     → Definitions with properties
Literature  → From papers/books
Method      → Techniques and processes
Question    → Research questions
```
**Full explanation:** [ZETTELKASTEN_GUIDE.md](ZETTELKASTEN_GUIDE.md) → Note Types

## 🎨 Customization

### Change Note Directory
```elisp
(setq org-roam-directory "~/your/path/roam")
```
**More options:** [INSTALLATION.md](INSTALLATION.md) → Customization

### Add Custom Note Type
```elisp
;; In init-roam.el
("x" "your-type" plain "template" ...)
```
**Examples:** [INSTALLATION.md](INSTALLATION.md) → Customization

### Change Keybindings
```elisp
(define-key org-roam-mode-map (kbd "C-c z") #'org-roam-node-find)
```
**More examples:** [INSTALLATION.md](INSTALLATION.md) → Customization

### Disable Features
```elisp
;; Comment out in init.el
;; (require 'init-org-appearance)
```
**Configuration levels:** [INSTALLATION.md](INSTALLATION.md) → Configuration Options
